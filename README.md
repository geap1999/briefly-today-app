# Briefly Today

**Briefly Today** is a mobile app that delivers interesting facts and notable events that happened on this date in history, every day. In addition to historical facts, the app features a "Scoop of the Day" section highlighting real-time, current events (https://play.google.com/store/apps/details?id=com.geap.brieflytoday).

## Features

- **On This Day:** Discover historical facts and events that happened on each day of the year.
- **Scoop of the Day:** Stay updated with a daily highlight of current, real-time events.
- **Simple & Clean UI:** Easy-to-use interface for quick daily insights.

## Technology

- Built with [Expo](https://expo.dev/) for fast, cross-platform React Native development.
