# Briefly Today

**Briefly Today** is a mobile app that delivers interesting facts and notable events that happened on this day in history, every day. In addition to historical facts, the app features a "Scoop of the Day" section highlighting real-time, current events.

## Features

- **On This Day:** Discover historical facts and events for each day of the year.
- **Scoop of the Day:** Stay updated with a daily highlight of current, real-time events.
- **Simple & Clean UI:** Easy-to-use interface for quick daily insights.

## Technology

- Built with [Expo](https://expo.dev/) for fast, cross-platform React Native development.

## Getting Started

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd briefly-today-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npx expo start
   ```
